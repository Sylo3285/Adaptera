Thank you for your interest in contributing to Adaptera! We welcome contributions from the community to make local-first LLM orchestration easier and more accessible.

## How to Contribute

1. **Bug Reports**: If you find a bug, please open an issue with a clear description and steps to reproduce.
2. **Feature Requests**: Have an idea for a new feature? Open an issue to discuss it first.
3. **Pull Requests**:
   - Fork the repository.
   - Create a new branch for your feature or fix.
   - Write tests for your changes.
   - Ensure all code follows basic formatting guidelines.
   - Submit a pull request!

## Development Setup

We recommend using `Miniforge` or `Conda` for environment management.

```bash
# Clone the repository
git clone https://github.com/Sylo3285/Adaptera
cd Adaptera

# Install dependencies in editable mode
pip install -e ".[dev]"
```

## Code of Conduct

Please be respectful and helpful to others in the community.
